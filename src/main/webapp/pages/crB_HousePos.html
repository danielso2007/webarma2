Grupo patrulha no interior de prédio:

Descrição: 
Este script faz com que o grupo faça patrulhas entre os prédios ou fique somente no prédio.
Detalhes: 
Mova unidade a posição da casa aleatória dentro de 10m
0 = [este] execVM "crB_scripts \ crB_HousePos.sqf";
Mova a unidade de posição da casa aleatória dentro 50m
0 = [isto, 50] execVM "crB_scripts \ crB_HousePos.sqf";
Unidade de patrulha para cargos de casas aleatórias dentro de 50m, com um máximo de espera de tempo de 300 segundos entre as posições
0 = [este, 50, true] execVM "crB_scripts \ crB_HousePos.sqf";
Unidade de patrulha para cargos de casas aleatórias dentro de 50m, com um máximo de espera de tempo de 600 segundos entre as posições
0 = [este, 50, é verdade, 600] execVM "crB_scripts \ crB_HousePos.sqf";

Parâmetros: 
- o objeto (Grupo,unidade).
- o raio de patrulha/busca de prédio.
- True o grupo irá patrulhar, false fica parado no primeiro prédio.
- o tempo de espera aleatório máximo em segundos entre patrulhas.
Como Funciona: 
-Coloque o script no INIT da unidade ou no lider do grupo.
SCRIPT: 
0 = [this, 50, true, 600] execVM "crB_scripts\crB_HousePos.sqf";
Arquivo: crB_HousePos.sqf
