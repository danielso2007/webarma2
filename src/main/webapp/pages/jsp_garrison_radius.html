SOLDADOS NO INTERIOR DO PRÉDIO:

Descrição: 
O roteiro coloca os homens no interior das casas em janelas, portas e nos telhados e varandas. Os homens podem ter uma probabilidade de presença. 
Ele atualmente só funciona para edifícios no Takistan, Shapur, Deserto, Zargabad, e alguns no Proving Grounds.
Detalhes: 
garrison_01 = ["marker_name",radius,#,%_of_positions,min_skill,max_skill,faction#,"marker_name_2",janela#,porta#,telhado#,varanda#] execVM "jsp_garrison_radius.sqf";
-"marker_name" - o marcador em torno do qual você deseja que apareça os soldados.
-radius - o raio em torno do marcador que você quer edifícios a ser ocupado.
-_of_buildings% - a percentagem de edifícios que você deseja ser ocupado. 0,1 = 10%: 0,5 = 50%. Qualquer número 0-1 devem trabalhar.
-_of_positions% - o percentual de posições que pretende ser ocupado. 0,1 = 10%: 0,5 = 50%. Qualquer número 0-1 deve funcionar.
-min_skill - a quantidade mínimo de habilidade para as unidades.
-max_skill - a quantidade máxima de habilidade para as unidades.
-faction# - o número para a facção que você deseja criar (encontrado abaixo).
-"marker_name_2" - define onde as unidades irá aparecer antes de serem colocados nos edifícios.(tente não colocar este marker para aparecer direto no prédio).
janela# - define se as unidades irá aparecer em janelas (0 = Off / Não, 1 = On / Sim)
porta# - define se as unidades irá aparecer em portas (0 = Off / Não, 1 = On / Sim)
telhado# - define se as unidades irá aparecer no telhado (0 = Off / Não, 1 = On / Sim)
varanda# - define se as unidades irá aparecer nas varandas (0 = Off / No, 1 = On / Sim)
